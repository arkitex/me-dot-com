PONTIFLEX APPLEADS

Pontiflex Appleads is a pioneering mobile advertising solution available for iOS, Android, and Windows Phone. I developed and maintained all the client side SDK code, working with Javascript, Objective-C, Java, and C#. I developed our Windows Phone and Windows 8 SDKs from the ground up, while working closely to add features to the Android and iOS side. I worked closely with the mobile business development team to support developers using our ad SDK, and ran the customer support site on my own.

ART INTERACTIVE

The future of museum interaction. Art Interactive is a fast, fluid iOS app that allows museum exploration from the top down. Pull back to see all the museums in New York, and drill down to see each artwork in each museum. As the iOS development lead, I work closely with the founder and designer to make their ideas reality. LAST RIDE IN

Last Ride In is a transit framework that Matt Conlen and I built to easily view timetables on the desktop and mobile web. Built off of node-gtfs, it can pull gtfs files published by any transit authority and place them into a Mongo database. The data contained within is then served up using a server running on Node.js. The source is available on github.

MICROSOFT COUPON CLASSIFIER

I spent a summer in Beijing, China, interning at the Microsoft Advanced Technology Center. While there, I built a naive bayesian classifier to classify over 10,000 coupons into distinct categories. I built it from the ground up in C# using SQL Server and ASP.Net. It used a combination of different machine learning algorithms to increase the accuracy of the classifier.
